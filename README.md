
# Roll Dice Socket Go

This is a simple Roll Dice game implemented in Go, using sockets for singleplayer functionality.


## Requirements

Go 1.16 or higher


## Running Tests

To run tests, run the following command

```bash
  make run
  and
  just run index.html on /rolldice-web
```
